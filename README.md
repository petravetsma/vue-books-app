# Vue.js books list app


## Client


### Requirements

node.js 16+
npm 8+

### Project Setup

```sh
npm install
```

#### Compile and Hot-Reload for Development

```sh
npm run dev
```

#### Type-Check, Compile and Minify for Production

```sh
npm run build
```

#### Lint with [ESLint](https://eslint.org/)

```sh
npm run lint
```

## Server

### Requirements

yarn

### Project Setup

```sh
yarn
```

#### Start server

```sh
yarn run server
```
